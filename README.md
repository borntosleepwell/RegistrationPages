# TestPages

Aplikasi Android sederhana untuk halaman registrasi yang dibuat menggunakan Kotlin dan XML Layout.

## Deskripsi
Aplikasi ini merupakan contoh implementasi antarmuka pengguna (UI) sederhana untuk proses pendaftaran atau input data mahasiswa, yang mencakup input Nama dan NIM.

## Fitur
- **Halaman Registrasi**: Tampilan bersih menggunakan `ConstraintLayout`.
- **Input Nama**: Menggunakan `EditText` untuk memasukkan identitas nama.
- **Input NIM**: Menggunakan `EditText` dengan tipe input angka (`number`).
- **Tombol Register**: Tombol untuk memproses pendaftaran.
- **Edge-to-Edge**: Mendukung tampilan layar penuh yang modern.
- **Journal Page**: User dapat menambahkan journal ke dalam aplikasi.

## Teknologi yang Digunakan
- **Bahasa**: Kotlin
- **UI Framework**: XML (AppCompat)
- **Tooling**: Android Studio
- **Komponen**: 
  - View Binding / Layout Resource
  - ConstraintLayout
  - Edge-to-edge support

## Cara Menjalankan
1. Clone repositori ini.
2. Buka proyek menggunakan **Android Studio**.
3. Tunggu proses Gradle Sync selesai.
4. Hubungkan perangkat Android atau gunakan Emulator.
5. Klik tombol **Run** (ikon play hijau).
